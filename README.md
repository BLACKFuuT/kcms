KCMS is a Django-based backend project built to practice real-world backend engineering concepts using **Django** and **Django REST Framework (DRF)**.  
The project intentionally focuses on architecture, permissions, authentication, and API design rather than frontend styling.

This repository demonstrates how traditional Django (HTML views) and DRF APIs can coexist cleanly in the same codebase.

---

## Project Goals

- Build a production-style Django backend
- Understand Django vs DRF mental models
- Implement JWT authentication correctly
- Apply object-level permissions
- Design scalable and maintainable APIs
- Optimize database queries for performance

---

## Tech Stack

- Python
- Django
- Django REST Framework
- JWT Authentication
- SQLite (development)
- Postman (API testing)
